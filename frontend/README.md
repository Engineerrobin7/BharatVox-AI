# Frontend Application

This directory contains the frontend for the BharatVox AI application. It is built using vanilla JavaScript, HTML, and CSS, and interacts with the backend API to provide voice detection functionality.

## Files

- `index.html`: The main HTML file for the application.
- `app.js`: The core JavaScript file containing the application logic.
- `styles.css`: The stylesheet for the application.

## Usage

To run the frontend, simply open `index.html` in your web browser. Ensure the backend server is running for full functionality.
